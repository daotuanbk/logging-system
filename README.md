# Logging System Using ES and Kibana

## Introduction
Logging System Using ES and Kibana

## Requirements
NodeJS
MongoDB
npm
homebrew

## Getting Started

1. Install Elasticsearch
brew tap elastic/tap
brew install elastic/tap/elasticsearch-full

2. Install kibana
brew tap elastic/tap
brew install elastic/tap/kibana-full

3. Run MongoDB

4. Run Elasticsearch:
elasticsearch

5. Run Kibana
kibana

3. Start project: npm run start
